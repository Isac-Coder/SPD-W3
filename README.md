# SPD-W3

## Project Title
Festival Gastronómico Nocturno - Landing Page

## Brief Description
A responsive landing page for the Nocturnal Gastronomic Festival, featuring sections for upcoming events, festival information, experiences, and a gallery. Built with HTML5, CSS3, and designed for mobile, tablet, and desktop.

## Technologies Used
- HTML5 (semantic elements like header, main, section, nav, footer)
- CSS3 (Flexbox, Grid, media queries, animations)
- Fonts: Google Fonts (Bebas Neue, Playfair Display, Poppins)

## Project Structure
- `index.html`: Main HTML file with semantic structure including header, nav, main sections (hero, upcoming events, about, food zones, testimonials, gallery), and footer
- `css/styles.css`: Comprehensive stylesheet with CSS variables, Flexbox for header/hero, Grid for galleries/cards, responsive media queries, and animations
- `assets/`: Folder containing images (Logo.png, hero-fondo.png)
- `README.md`: Project documentation in English

## How to Run Locally
1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in a web browser, or use a local server:
   - Run `python -m http.server 8000` in the terminal from the project root.
   - Open `http://localhost:8000` in your browser.
